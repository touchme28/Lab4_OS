# Lab4_OS

Программа с системными вызовами, обеспечивающими создание процессов.

## Описание

Программа 1 (lab4_1.c) :
- Исполняет роль дочерней программы.
- При запуске принимает аргументы командной строки, а затем в цикле выводит каждый аргумент на экран с задержкой в одну секунду.
- Выводит на экран свой идентификатор и идентификатор процесса-родителя.
- Формирует код завершения, равный 5.
- Выводит полученные переменные на экран.
  
Программа 2 (lab4_2.c) :
- Исполняет роль родительского процесса и запускает программу 1 в качестве дочернего процесса с помощью вызовов fork() и execle().
- При запуске в терминале получает аргументы командной строки и выводит их на экран.
- Выводит на экран идентификатор процесса-родителя, свой идентификатор и идентификатор дочернего процесса.
- Формирует набор параметров для передачи в дочерний процесс через функцию ехесle(). (в данном случае идет передача 3 аргументов командной строки программы 2 программе 1 и программа 1 выводит их на экран / также создается переменная окружения (env), которая передается через функцию ехесle() потомку, а потомок выводит на экран.)
- Ожидает завершения дочернего процесса, проверяя событие завершения каждую половину секунды, а по завершению дочернего процесса вывести на экран код его завершения.
